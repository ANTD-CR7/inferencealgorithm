User Guide and Technical Reference

User Guide

  1. Select a network from the dropdown.
  2. Choose a query variable.
  3. Click nodes to set evidence (TRUE → FALSE → clear).
  4. Run inference and review results.

API Endpoints

GET /api/networks – Returns network metadata.

POST /api/inference – Runs inference and returns probabilities.
